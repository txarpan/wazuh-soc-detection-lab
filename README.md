# Detection Engineering & SOC Lab using Wazuh SIEM (Dockerized)

## Overview
This project is a SOC lab built using Wazuh SIEM to practice detection engineering, log analysis, and attack simulation.

## Features
- Custom rule creation (Wazuh)
- SIEM log analysis
- Attack simulation (planned)
- SOC troubleshooting workflow

## Tech Stack
- Wazuh SIEM
- Docker
- Linux
- Bash
